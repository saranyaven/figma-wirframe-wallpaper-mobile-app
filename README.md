# figma-wirframe-wallpaper-mobile-app## Project Overview
This is a mobile app wireframe for a **Wallpaper Application** created as part of the Google UI/UX Design Professional Certificate.

## Design Tool
* Figma

## Key Features
* User-friendly navigation
* Categories for different wallpaper styles
* Preview and set wallpaper options
